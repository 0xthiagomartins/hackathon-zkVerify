# Sprint 1: Configuração e Layout Básico

## Objetivos

Nesta sprint inicial, o objetivo é configurar o ambiente de desenvolvimento e criar a estrutura básica da interface de usuário para o aplicativo DeGym zkVerify. Focaremos na criação de um layout funcional que servirá como base para as funcionalidades que serão implementadas nas próximas sprints.

## Metas Específicas

1. **Configuração do Projeto**
   - Inicializar o projeto Next.js com TypeScript
   - Configurar TailwindCSS para estilização
   - Configurar variáveis de ambiente necessárias

2. **Layout Básico**
   - Criar componente de cabeçalho com logo DeGym
   - Implementar estrutura básica da página de check-in
   - Criar componente de seleção de academia (dropdown)

3. **Componentes Iniciais**
   - Implementar componente de container para o mapa (placeholder inicialmente)
   - Criar botões de ação (obter localização, fazer check-in)
   - Implementar componentes de feedback visual para os diferentes estados

## Critérios de Aceitação

- Projeto Next.js configurado e funcionando corretamente
- TailwindCSS aplicado e funcionando
- Layout responsivo funcionando em desktop e mobile
- Componentes visuais básicos implementados e estilizados
- Estrutura de diretórios organizada seguindo as melhores práticas do Next.js
